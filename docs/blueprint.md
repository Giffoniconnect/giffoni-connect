# **App Name**: Giffoni Connect

## Core Features:

- Case Status Tracking: Allow clients to view the current status of their cases.
- Document Access: Provide secure access to relevant legal documents.
- Secure Messaging: Enable secure communication between clients and the legal team.
- Invoice Review: Allow clients to review and download invoices.
- Personalized legal recommendations: Based on case documents and customer profile data, the tool suggests potentially useful legal documents (templates) to help with the legal issue

## Style Guidelines:

- Primary color: Navy blue (#2E3A87) to convey professionalism and trust.
- Background color: Light gray (#F0F4F7) for a clean and modern feel.
- Accent color: White (#FFFFFF) to highlight important elements and create contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and objective look.
- Use professional and clear icons to represent different actions and categories.
- Clean and structured layout to ensure ease of navigation and readability.
- Subtle animations for transitions and feedback to enhance user experience.
